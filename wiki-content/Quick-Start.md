## Troubleshooting

### Data Not Saving?
- **Check browser localStorage** is enabled
- **Disable private/incognito mode** (data won't persist)
- **Clear browser cache** and reload
- **Check storage quota** (should be 5MB+)

### App Not Loading?
- **Check internet connection** (required for first load)
- **Try a different browser**
- **Clear cache and reload**
- **Disable browser extensions** that might interfere

### Charts Not Showing?
- **Log at least 2 days of data** to see trends
- **Refresh the page** if chart doesn't appear
- **Check browser console** for JavaScript errors

### Install Button Not Appearing?
- **Already installed?** Check your apps/home screen
- **Using Safari on iOS?** Use the Share â†’ Add to Home Screen method instead
- **Using Firefox?** PWA install support is limited; use Chrome/Edge

### Lost Your Data?
If you accidentally signed out or cleared data:
- âŒ **Unfortunately, data cannot be recovered** (it's stored locally only)
- âœ… **Export data regularly** to prevent loss
- âœ… **Consider syncing exports** to cloud storage (Google Drive, Dropbox, etc.)
